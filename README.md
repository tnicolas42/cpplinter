# cpplinter

## installation
clone the repository
```bash
git clone git@github.com:tnicolas42/cpplinter
```
source the file in `~/.zshrc`
```zsh
source PATH_TO_CPPLINTER/alias.zsh
```

## usage
use the linter with specifics rules
```bash
cpplint <cppfiles>
```

use with all rules
```bash
cpplint_allrules <cppfiles>
```
